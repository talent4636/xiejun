b2c.order.check_cost 确认订单价格统计接口

API应用级输入参数

member_id int Y 会员id
accesstoken string Y 会员登录返回的 accesstoken
area_id int N 购物车确认收货地区ID
shipping_id int N 配送方式
is_protect string N 是否支持物流保价 true false
cur string N 支付货币（默认为后台设置的默认货币）
is_tax string N 是否需要发票
tax_type string N 发票类型，不同发票类型为不同的费率
dis_point int N 抵扣积分
