b2c.brand.get_brand_detail 获取品牌数据(根据品牌排序查询列表)

API应用级输入参数

brand_name string N 品牌名称，可根据品牌名称查询品牌基本数据
brand_id int N 品牌ID，可根据品牌ID查询品牌数据, 品牌ID可以多个
page_no int N 页码 默认为1
page_size int N 分页大小 默认10
