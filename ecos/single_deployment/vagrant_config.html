VagrantBox详细参数(Box包选择手册)

ecstore-lnmp-0.0.2.box

  • 应用以及配置
    项目 内容
    centos 6.5
    libmcrypt 2.5.7
    libxml 2.9.0
    freetype 62.3.5
    gd 2.0.35
    mysql 5.6.16
    nginx 1.5.10
    php 5.3.28
    zend guard loader php-5.3-linux-glibc23-x86_64
    mysql用户名密码 用户名：root，密码无
    linux用户名密码 root用户和vagrant用户，密码均为vagrant
    安装目录 /usr/local/webserver（php/mysql/nginx/freetype/gd）
    启动脚本 /etc/init.d/(php-fpm|nginx|mysql)
    php配置文件 /usr/local/webserver/php/php.ini
    php-fpm配置文件 /usr/local/webserver/php/etc/php-fpm.conf
    nginx配置文件 /usr/local/webserver/nginx/conf/nginx.conf
    mysql配置文件 /etc/my.cnf
    共享目录 /vagrant/
    www目录 /vagrant/ecstore/
  • 端口映射关系
    应用 端口号 映射端口号
    ssh 22 2222
    nginx 80 8000
    mysql 3306 33060
    php-fpm 9000 -

commerce_b2b2c-lnmp-with-prism-0.0.2.box

  • 应用以及配置
    项目 内容
    centos 6.5
    libmcrypt 2.5.7
    libxml 2.9.0
    freetype 62.3.5
    gd 2.0.35
    mysql 5.6.16
    nginx 1.5.10
    php 5.4.28
    prism 1.0.24
    mongodb(仅为prism使用) 3.0.2
    rabbitmq-server 3.5.1
    zend guard loader php-5.4-linux-glibc23-x86_64
    mysql用户名密码 用户名：root，密码无
    prism用户名密码 用户名：admin，密码：admin
    linux用户名密码 root用户和vagrant用户，密码均为vagrant
    安装目录 /usr/local/webserver（php/mysql/nginx/freetype/gd）
    启动脚本 /etc/init.d/(php-fpm|nginx|mysql|prism|mongodb|rabbitmq-server)
    php配置文件 /usr/local/webserver/php/php.ini
    php-fpm配置文件 /usr/local/webserver/php/etc/php-fpm.conf
    nginx配置文件 /usr/local/webserver/nginx/conf/nginx.conf
    mysql配置文件 /etc/my.cnf
    共享目录 /vagrant/
    www目录 /vagrant/
  • 端口映射关系
    应用 端口号 映射端口号
    ssh 22 2222
    nginx 80 8000
    prism 8080 8001
    mongodb 27017 -
    rabbitmq-server(web)(预占用) 15672 8002
    mysql 3306 33060
    php-fpm 9000 -

內容目录

    • ecstore-lnmp-0.0.2.box
    • commerce_b2b2c-lnmp-with-prism-0.0.2.box
